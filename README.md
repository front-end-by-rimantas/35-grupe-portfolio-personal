# React portfolio

_35 group personal website project_

This project is for educational porpuses only. Pull request are welcome, but priority for project authors! Thank you for your cooperation!

- Github pages
- multi-page
- content rendering with JS
- carousel
- fontawesome icons
- CSS animations
- CSS import
- JS import-export
